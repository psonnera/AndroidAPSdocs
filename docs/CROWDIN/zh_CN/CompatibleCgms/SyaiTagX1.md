# Syai X1


## 使用X1与Syai Tag应用

-   从<https://play.google.com/store/apps/details?id=com.syai.tag>下载并安装该apk文件。

-   启动X1传感器

- 在[ConfigBuilder，BG源](#Config-Builder-bg-source)中选择Syai Tag。

请在Syai应用中启用广播功能：

1. 选择 Profile
2. Syai健康生态系统
3. 点击"立即与AAPS连接"，同意第三方数据传输协议
4. 启用"与AAPS共享数据"功能
