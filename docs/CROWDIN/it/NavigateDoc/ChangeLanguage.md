# Cambia lingua

::: info
The language selection can be found bottom right of the screen. Fai clic sulla barra in basso per aprire il menu di selezione lingua.
:::


![Apri menu lingua](../images/documentation_language_menu.png)

::: warning
When changing language and/or version, you might get a 404 - Page not found, if the language or version page doesn't exist. Questo accade di frequente quando cambi versione ma non dovrebbe accadere quando cambi lingua: la pagina sarà in inglese se non è ancora tradotta.
:::


Sei invitato ad unirti ai traduttori e revisori [Crowdin](https://crowdin.com/project/androidapsdocs).
